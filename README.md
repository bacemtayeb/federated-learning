# Federated Learning
Federated topic modeling to identify common topics in news articles across distributed sources, such as news outlets or regional branches.
#### Steps to reproduce:
###### 1. Run data prepration script. It will create new directory with synthetic clients.
###### 2. Run server script.
###### 3. Run in another session the client script for training.

###### data source: https://www.kaggle.com/datasets/rmisra/news-category-dataset
